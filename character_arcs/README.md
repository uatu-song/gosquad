# Character Arcs - Go Squad Series (Books 1-8)

Character development documents tracking arcs across the complete 8-book series.

---

## Go Squad Core Members

- **Ahdia_Arc_Tracker.md** - Protagonist across all 8 books (Hyper Temporalist, designed singularity)
- **Ruth_Arc_Tracker.md** - Team leader after Firas dies (Nightingale)
- **Tess_Arc_Tracker.md** - Tech/Stealth specialist, Chief Whitford's daughter (Gloom Girl)
- **Ryu_Arc_Tracker.md** - CADENS handler, AR-Ryu split in Book 4

## Go Squad Extended Members

- **Ben_Arc_Tracker.md** - Ex-Marine, combat specialist (Night Knight)
- **Victor_Arc_Tracker.md** - Community organizer, connector character (Crimson Sable)
- **Leah_Arc_Tracker.md** - Speed specialist, white moderate growth arc (Battlea)
- **Korede_Arc_Tracker.md** - Teen recruit Book 3, resistance leader Book 4

## Key Supporting Characters

- **Diana_Arc_Tracker.md** - Ahdia Prime, transcends in Book 4, returns Book 8
- **Leta_Arc_Tracker.md** - Tess's girlfriend (Korede's sister), dies Book 2
- **Firas_Arc.md** - Go Squad founder, "dies" Book 1, returns Book 7

## Institutional Characters

- **Harriet_Bourn_Arc_Tracker.md** - CADENS Director, defects with wake protocol
- **Bentley_Mack_Arc_Tracker.md** - Military liaison, defects with Bourn
- **FAERIS_Arc_Tracker.md** - Mother FAERIS and drones (AI collective)

## Primary Antagonists

- **Harding_Kain_Arc_Tracker.md** - Human villain, billionaire → President
- **Eidolon_Arc_Tracker.md** - Fear entity, serves Bellatrix → evolves
- **The_Collector_Arc_Tracker.md** - Type III being, Book 5 temptation villain
- **The_Predecessor_Arc_Tracker.md** - Failed species, Books 6-7 antagonist

## Cosmic Characters
