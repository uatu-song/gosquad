### Dialogue-Heavy Chapters (Maintain in Future Books)

### Dialogue Rules
1. **Keep it snappy** - Most lines under 150 characters
2. **Use contractions** - "won't", "she'd", "they're" (natural speech)
3. **Break grammar rules** - Fragments, interruptions, trailing off
4. **Show character voice** - Each character has distinct patterns
5. **Action beats over tags** - "He ducked" not "he said, ducking"

## ðŸŽ¯ Genre-Specific Guidelines

### Superhero/Action Elements
1. **Powers:** Show, don't explain
2. **Fight choreography:** Short, clear, visceral
3. **Team dynamics:** Banter + tension
4. **Moral complexity:** Shades of gray
5. **Stakes:** Personal + global

### Action Scene Structure
**From Book 1's Climax (Chapter 22):**
1. Setup in 1-2 sentences
2. Rapid action beats (fragments OK)
3. Character reactions mid-action
4. Sensory details (sight, sound, impact)
5. Consequences immediate
6. No long descriptions during action

### Example Structure:
